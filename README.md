🚀 Product Growth Intelligence Copilot

An end-to-end Business Intelligence and Product Analytics project that transforms raw business data into actionable insights using SQL, Python, Tableau, and GitHub.

This project demonstrates how a Data Analyst can build a complete analytics workflow—from data extraction and transformation to interactive dashboard development—answering key business questions around revenue, customer behavior, conversion funnels, and payment performance.

⸻

📌 Project Objectives

* Analyze business performance using SQL-based analytics.
* Design executive dashboards for business stakeholders.
* Measure customer acquisition, retention, and repeat purchase behavior.
* Identify revenue drivers across products, states, and payment methods.
* Build a reusable analytics pipeline suitable for Business Intelligence reporting.

⸻

🛠️ Tech Stack

* SQL (SQLite) – Data extraction, joins, aggregations, KPI calculations
* Python (Pandas) – Data preprocessing and CSV generation
* Tableau Public – Interactive dashboards
* Git & GitHub – Version control and project documentation
